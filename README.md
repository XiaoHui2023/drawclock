# drawio_to_clock_tree

从 draw.io 时钟拓扑图生成时钟树相关输出的本地命令行工具。

## 命令行参数

运行：`python src`

| 长参数 | 短参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| `--input` | `-i` | 文件路径 | 必填 | draw.io 源文件（`.drawio` 或 `.xml`） |
| `--output` | `-o` | 路径 | | 输出目录；未指定时写入当前工作目录 |
